# Homestyler 地台界面完整分析

## 1. 界面概述 🎯

**界面名称**: Platform Customization Interface (地台定制界面)

**界面用途**: 
- 设计和创建室内地台（抬高的地面区域）
- 应用于榻榻米、休息区、展示区等场景
- 配置地台高度、边缘造型、材质
- 集成储物功能

**技术实现**: 3D参数化建模 + 挤压造型

## 4. 地台造型系统 📐

### 4.1 基础形状

**预设形状**:
- **矩形地台**: 最常见，适合规则空间
- **圆形/椭圆地台**: 适合休闲区
- **L型地台**: 适合转角空间
- **U型地台**: 围合式设计
- **自由形状**: 根据空间定制

### 4.2 边缘处理

**边缘类型**:
```javascript
{
    straight: "直角边缘",      // 90度直角
    rounded: "圆角边缘",       // 圆弧过渡
    chamfered: "倒角边缘",     // 斜切边缘
    curved: "曲线边缘",        // 自由曲线
    stepped: "阶梯边缘"        // 多级台阶
}
```

**踢脚线设计**:
- 高度: 50-100mm
- 材质: 木质/金属/石材
- 造型: 简约/欧式/现代

## 5. 参数化地台 ⚙️

### 5.1 核心参数

```javascript
// 地台参数配置
{
    // 尺寸参数
    length: Number,        // 长度 (mm)
    width: Number,         // 宽度 (mm)
    height: Number,        // 高度 (mm) 通常100-500mm
    
    // 位置参数
    positionX: Number,     // X坐标
    positionY: Number,     // Y坐标
    rotation: Number,      // 旋转角度
    
    // 边缘参数
    edgeType: String,      // 边缘类型
    cornerRadius: Number,  // 圆角半径 (mm)
    chamferSize: Number,   // 倒角尺寸 (mm)
    
    // 功能参数
    hasStorage: Boolean,   // 是否含储物
    hasLighting: Boolean,  // 是否含灯带
    hasDrainage: Boolean   // 是否含排水（浴室地台）
}
```

### 5.2 高度分级

**常见高度规格**:
- **低地台**: 100-200mm（装饰性为主）
- **中地台**: 200-350mm（兼具储物功能）
- **高地台**: 350-500mm（榻榻米标准高度）


## 7. 功能集成 🔧

### 7.1 储物功能

**储物类型**:
- **上翻式**: 整体顶板上翻
- **抽屉式**: 侧面抽屉
- **格子式**: 分隔储物格
- **隐藏式**: 暗门设计

**储物参数**:
```javascript
{
    storageType: String,      // 储物类型
    drawerCount: Number,      // 抽屉数量
    drawerHeight: Number,     // 抽屉高度
    dividerCount: Number      // 分隔数量
}
```

### 7.2 照明系统

**灯光布置**:
- **边缘灯带**: 沿地台边缘
- **底部灯带**: 悬浮效果
- **嵌入式筒灯**: 顶面照明
- **氛围灯**: 装饰性灯光

**灯槽设计**:
```javascript
{
    lightPosition: String,    // 灯光位置 (edge/bottom/top)
    lightType: String,        // 灯光类型 (strip/spot/ambient)
    grooveWidth: 30-50mm,     // 灯槽宽度
    grooveDepth: 20-40mm      // 灯槽深度
}
```

### 7.3 排水功能 (浴室地台)

```javascript
PlatformDrainage: "Platform drainage"
```

**排水设计**:
- 坡度设计: 1-2%坡向排水口
- 排水槽: 线性排水
- 防水层: 防水材料处理
